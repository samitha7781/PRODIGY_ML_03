# Cats vs Dogs Image Classification using SVM

## Overview
This project implements a Support Vector Machine (SVM) model to classify images of cats and dogs. The model is trained on labeled image data and predicts whether an image belongs to a cat or a dog category.

Developed as part of the Prodigy InfoTech Machine Learning Internship.

---

## Objective
To build an image classification system using Support Vector Machine (SVM) that can accurately distinguish between cat and dog images.

## Project Workflow

### 1. Data Collection
- Loaded cat and dog image datasets.

### 2. Image Preprocessing

- Color conversion
- Feature extraction

### 3. Model Training
- Trained an SVM classifier on processed image data.

### 4. Model Evaluation
- Accuracy Score
- Classification Report
- Confusion Matrix

### 5. Prediction and Visualization
- Predicted image classes.
- Visualized actual and predicted labels.
- Displayed sample classification results.
